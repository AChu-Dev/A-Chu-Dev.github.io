Python

I used Python initally at College to text compression tasks which would involve using a function to compress & decompress text inputted via strings. When I did this task I did not know if I was allowed to use the libary GZIP at the time so with help from a tutor we created a compression algorithm.

So the concept is based around using enumerate and creates a unique identifier based on each word with different identifiers if there is capitalisation or punctuation. It then saves these lists under two different files one contains a list with the positions of ach word using the identifiers which are incrementing numbers. The other file is a series of numbers which contains the words and the identifier in a (#*2) list. The unzipping function reverses the process to create the string and would recreate the original string by cross referencing the lists.
